使用提示 Tips

會員操作：到「會員」分頁可手動延長到期日、停用 access code、修改 email。

取消訂閱處理：客戶申請取消後會出現在「取消申請」分頁。處理流程：(1) 到綠界後台停止扣款；(2) 在「會員」中將該 access code 狀態改為 cancelled。

系統異常：若 KV 未啟用，會員資料無法即時更新，需手動編輯 Vercel 環境變數 VALID_ACCESS_CODES。

資料備份：建議每月匯出一次 KV 資料備份（手動：到「會員」分頁 → 下方匯出 CSV 按鈕）。
